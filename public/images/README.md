# Document assets

Placeholder images used by the digital document viewer:

- **logo.svg** – Shown in the receipt header. Replace with your brand logo (e.g. export from production or use PNG).
- **banner.svg** – Hero image below the document title. Replace with a lifestyle/banner image (e.g. `banner.jpg`).
- **avatar.svg** – Silhouette in the thank-you block. Replace with your avatar graphic if desired.

Use the same filenames so the app keeps working without code changes. For PNG/JPG, replace the file and update `src/config/assets.ts` to point to the new extension (e.g. `logo.png`).
